# Covid19Tracker
Latest COVID-19 Stats, English & Bengali News, Weather details for Current Location and All Country with Cool UI


## Covid19Tracker

* Firebase

* OkHttp
* StatusBarUtil
* Glide
* Circle ImageView
* Dynamic Toast
